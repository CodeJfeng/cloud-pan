# 企业级的云盘系统
**一个基于现代技术栈构建的企业级文件存储与共享平台，提供安全可靠的文件管理、分享和协作功能。**

本项目是一个基于Spring Boot + Vue.js + MyBatis-Plus + Redis + 阿里云OSS的企业级云盘系统。旨在为企业提供一个安全、高效、便捷的私有云存储解决方案。系统支持多用户注册与登录、文件的上传下载、在线预览、文件分享、文件夹管理以及权限控制等核心功能，满足企业内部文件协同办公的需求。

## 🌟 项目特色
- 企业级架构：采用前后端分离架构，支持高并发访问
- 安全可靠：多重安全机制保障企业数据安全
- 高性能：Redis缓存优化，提升系统响应速度
- 云存储集成：无缝集成阿里云OSS，支持海量文件存储
- 便捷分享：灵活的文件分享机制，支持企业内部协作

## 🛠️ 技术栈
### 后端技术
- Spring Boot 2.7+ - 后端核心框架
- MyBatis-Plus 3.5+ - 数据持久层框架
- Redis 5.0+ - 缓存与会话管理
- MySQL 8.0 - 主数据库
- JWT - 身份认证机制
- 阿里云OSS - 对象存储服务
- Maven - 项目构建管理
### 前端技术
- Vue 3 - 前端核心框架
- Element Plus - UI组件库
-Axios - HTTP请求库
- Vue Router - 路由管理
- Vuex/Pinia - 状态管理

## 主要功能及难点

### 用户模块：
#### 用户功能
- 提供用户注册和登录功能，支持JWT进行认证。
- 实现了用户信息的管理与维护。
- 用户的登入、登出、忘记密码、修改密码、查询用户信息
- 封装统一的登录校验器，验证登录信息
#### 难点
- 怎样实现用户名唯一的幂等性注册接口
  - 通过数据库的username**字段唯一性**，查询数据库是否存在该username的信息，解决了重复注册的问题
- 怎样实现全局统一的登录拦截器
  - 使用**aop Aspect**切面的形式，当执行module下的方法，进入切面方法
- redis缓存登录的单机的，如何在集群模型下实现登录的信息共享
  - 该方式采用**JWT Token**的方式实现时效性以及分布式问题，通过解析token判断
  - 也可使用nginx session粘贴功能实现
#### 学习收获
- 养成良好的业务级接口的编码习惯
- 养成良好的单元测试习惯
- 学习使用切面思想实现统一的业务接口增强逻辑
- 了解幂等相关概念、提升接口的稳定性
#### 目前待升级的问题
- 目前JWT实现的是缓存下的单机登录，当服务器是集群形式，该验证方法不能被其他集群所记录，导致用户验证失败。

### 文件管理：
- 支持文件的拖拽上传、断点续传和秒传功能。
- 提供文件的下载、删除、重命名和移动等基本操作。
- 支持多种文件类型的在线预览，如图片、视频、PDF和常见文档格式。
### 文件夹管理：
- 支持文件夹的创建、删除、重命名和移动。
- 实现了树状文件结构展示，方便用户浏览和管理。
### 文件分享：
- 用户可以创建私密分享链接，并设置过期时间和提取码。
- 支持将文件或文件夹分享给企业内部的其他用户或部门。
### 权限控制：
- 基于角色的访问控制（RBAC），管理员可以对不同用户的操作权限进行精细化管理。
- 支持设置文件夹的访问权限，实现不同部门或项目组之间的文件隔离。
### 回收站功能：
- 删除的文件会进入回收站，用户可以进行恢复或彻底删除。
